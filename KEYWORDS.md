# Keywords Documentation

## Answer: YES, there ARE keywords

The SurdMC Landing Page includes SEO meta keywords in the HTML file.

## Location

**File:** `public/index.html` (line 10)

## Current Keywords

```html
<meta name="keywords" content="minecraft, szerver, surdmc, magyar minecraft, minecraft server, modded, neoforge, mekanism, ender io, ae2, tech mods, building">
```

## Keyword List

The following keywords are currently defined for SEO purposes:

1. **minecraft** - Core game
2. **szerver** - Hungarian word for "server"
3. **surdmc** - Server brand name
4. **magyar minecraft** - Hungarian Minecraft
5. **minecraft server** - English term
6. **modded** - Indicates modded gameplay
7. **neoforge** - Mod loader platform
8. **mekanism** - Tech mod
9. **ender io** - Tech mod
10. **ae2** - Applied Energistics 2 mod
11. **tech mods** - Category descriptor
12. **building** - Gameplay style

## Purpose

These keywords help search engines understand the content and purpose of the website, improving SEO and discoverability for users searching for:
- Hungarian Minecraft servers
- Modded Minecraft servers
- Specific mods (Mekanism, Ender IO, AE2)
- NeoForge servers
- Tech and building focused servers

## Note on Meta Keywords

While modern search engines like Google don't heavily weight the `<meta name="keywords">` tag anymore, it's still used by:
- Some smaller search engines
- Internal site search tools
- As documentation for content strategy
- Legacy SEO purposes

The more important SEO factors currently implemented are:
- Title tags (✓)
- Meta descriptions (✓)
- Open Graph tags (✓)
- Structured data / JSON-LD (✓)
- Quality content (✓)
- Semantic HTML (✓)
